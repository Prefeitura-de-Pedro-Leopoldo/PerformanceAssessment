# Avaliação de Desempenho dos Servidores

Este repositório reúne formulários web para avaliação de desempenho dos servidores municipais de Pedro Leopoldo.

Inclui:

- `autoavaliacao.html` – Formulário de autoavaliação  
- `comissao.html` – Formulário de avaliação pela comissão  
- `servidor.html` – Formulário de avaliação final

Objetivo: garantir processo transparente, justo e eficiente de avaliação de desempenho.
